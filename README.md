# cybersecurity-beginner-project

A beginner-friendly cybersecurity toolkit built in Python. This project includes practical tools to help you learn core cybersecurity concepts hands-on.

---

## 📦 Tools Included

| Tool | File | Description |
|------|------|-------------|
| 🔑 Password Strength Checker | `password_strength_checker.py` | Analyzes password strength with score & tips |
| 🔤 Caesar Cipher | `caesar_cipher.py` | Encrypt/decrypt messages + brute force mode |
| 🌐 Port Scanner | `port_scanner.py` | Multi-threaded TCP port scanner |
| #️⃣ Hash Generator | `hash_generator.py` | MD5, SHA1, SHA256, SHA512 hash generator & verifier |

---

## 🚀 Getting Started

### Prerequisites
- Python 3.x installed → [Download Python](https://www.python.org/downloads/)

### Clone the Repo
```bash
git clone https://github.com/karansoni15-create/cybersecurity-beginner-project.git
cd cybersecurity-beginner-project
```

### Run the Tools
```bash
python password_strength_checker.py
python caesar_cipher.py
python port_scanner.py
python hash_generator.py
```

## 🧰 Tool Details

### 🔑 Password Strength Checker
Checks how strong your password is based on:
- Length
- Uppercase & lowercase letters
- Numbers
- Special characters

### 🔤 Caesar Cipher
A classic substitution cipher. Supports:
- Encryption with a custom shift key
- Decryption with the same key
- Brute force attack (tries all 25 shifts)

### 🌐 Port Scanner
Scans a target IP or hostname for open TCP ports. Features:
- Custom port range
- Multi-threaded for speed
- Displays open ports in real time

### #️⃣ Hash Generator
Generate and verify cryptographic hashes:
- MD5
- SHA1
- SHA256
- SHA512
